# Alex Original Website / Portfolio (Circa 2016)

🔗 https://alexandersuglio.github.io/subway-website-archive/

This project is one of the very first things I ever built when I started programming.

At the time, I wanted to highlight my passion for creativity—especially design, imagery, and visual storytelling. Every graphic on the site was created by me in Photoshop, and I spent countless hours dialing in layout, responsiveness, and positioning entirely on my own.

This site was less about best practices and more about experimentation—pushing what I could do, learning by trial and error, and seeing what was possible with code and design combined.

It includes:
- Custom-designed graphics (all original, Photoshop)
- Early attempts at responsive layout and positioning
- Hand-crafted animations
- My original resume from that time period

---

## Reflection

I still revisit this project from time to time.

It’s a snapshot of where I was—both personally and professionally—at the very beginning of my journey. Looking back on it helps me appreciate how far I’ve come, and it always gets me excited thinking about where I’ll be just a few years down the road.
